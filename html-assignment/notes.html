HTML Tag Glossary

⁢nav>
A tag used a define a set of navigation links.
⁢img>
A tag used to embed an image in a web page.
⁢ul>
A tag used to create an unordered list.
⁢header>
A tag used to define a header for a document or section.
⁢footer>
A tag used to define a footer for a document or section.
